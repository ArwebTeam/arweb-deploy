# Usage
<!-- usage -->
```sh-session
$ npm install -g arweb-deploy
$ arweb-deploy COMMAND
running command...
$ arweb-deploy (-v|--version|version)
arweb-deploy/0.0.0 linux-x64 node-v10.16.3
$ arweb-deploy --help [COMMAND]
USAGE
  $ arweb-deploy COMMAND
...
```
